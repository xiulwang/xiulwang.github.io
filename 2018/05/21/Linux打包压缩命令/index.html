Linux打包压缩命令

发表于 2018-05-21 | 分类于 Linux

概念

打包：将众多文件聚集在一起，变成一个
压缩：把一个大文件压缩成小文件
Linux中很多压缩算法针对一个文件进行操作，所以通常先打包再进行压缩

打包

以.tar结尾的tar包. man tar查看命令

1. c: create; f: files 产生新的包
tar -cf name.tar files/directories
tar -cf name.tar file1 file2

2. x: extract 解包
tar -xf name.tar

3. r: append 将新的文件增加到包中去
tar -rf name.tar newfile

4. u: update 更新替代原始文件
tar -uf name.tar targetfile

5. t: list 列出包中所有文件
tar -tf name.tar

压缩

在打包或者解包时可以同时进行压缩和解压

gzip压缩

tar使用z参数调用gzip

1. 先打包，在调用gzip压缩
tar -czf name.tar.gz files

2. 对应的解压缩的命令
tar -xzf name.tar.gz

3. 单独使用gzip
gzip -d name.gz

bzip2压缩

tar使用j参数调用bzip2

1. tar -cjf name.tar.bz2 files
2. tar -xjf name.tar.bz2
3. bzip2 -d name.bz2

compress压缩

tar使用-Z这个参数调用compress

1. tar -cZf name.tar.Z files
2. tar -xZf name.tar.Z
3. uncompress name.Z
